# HOW TO RECORD

1. Set Phone 1 on tripod for face shots — film ALL face parts first in one sitting
2. Screen record laptop via Cmd+Shift+5 — do the full onboarding + session flow
3. Screen record Phone 2 — do the tipping flow
4. For the magic moment: film the laptop screen WITH Phone 1 while tipping from Phone 2 — this shows the tip arriving in real-time from YOUR perspective
5. Edit in iMovie: lay face cam as base, cut to screens when you reference them
6. Speed up wallet creation to 2x
7. Add labels: "Creating wallet...", "Scanning QR...", "Live tip!"
8. Total target: 60-90 seconds
